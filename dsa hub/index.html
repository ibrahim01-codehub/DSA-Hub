Data Structures

Data structures are the containers. They decide how information is arranged in memory so it can be inserted, deleted, searched, or connected.

Algorithms

Algorithms are the instructions. They define the step-by-step process used to transform input into the correct output.

Example: a graph is a data structure. Breadth-first search is an algorithm that uses a graph to explore nodes level by level.
