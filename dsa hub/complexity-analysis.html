Complexity Analysis

What is complexity analysis??

Complexity Analysis is a way to measure how efficient your code is, without actually running it.

Types of complexity analysis

Time Complexity

Time Complexity is a measure of how the number of steps an algorithm takes grows as the input size (n) grows.



If I throw more data at my code, how much slower does it get?

Space Complexity

Space complexity measures the total amount of memory an algorithm needs to run, as a function of the input size (n). This includes both the space for the input itself and any extra (auxiliary) space the algorithm creates.

How much extra scratch paper does my code need when I give it more data?

Space complexity has an answer in terms of an "O"
